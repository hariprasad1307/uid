# Item List Form

## Overview

The **Item List Form** allows users to add items with name, description, and quantity fields, dynamically adding them to a list.

## Features

- **Form Inputs**: Fields for item name, description, and quantity.
- **Dynamic List**: Added items are displayed in a list.
- **State Management**: `useState` to manage form data and the item list.

## Technologies Used

- **React**: For state management and form handling.
- **CSS**: For layout and form styling.
